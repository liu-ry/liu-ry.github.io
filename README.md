My Personal Homepage

## Visitor Counter

This site uses GoatCounter to display:

- monthly page views
- all-time page views

Setup:

1. Create a site at `https://www.goatcounter.com/`
2. Enable `Allow adding visitor counts on your website` in the site settings
3. Set `analyticsConfig.goatcounterCode` in [script.js](/home/lry/src/liu-ry.github.io/script.js:162)

Notes:

- The site is still hosted on GitHub Pages; no custom backend is required
- Counts are based on page opens
- GoatCounter notes that counter responses may be cached for up to four hours
- Ad blockers may prevent some visits from being counted
